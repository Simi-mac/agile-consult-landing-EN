# Projeto

## Informações do projeto

Este repositório contém o site da Tribo Ágil.

## Como editar o código

Você pode editar localmente com seu IDE favorito clonando o repositório e instalando dependências.

Requisitos:

- Node.js e npm (recomenda-se usar `nvm`)

Passos rápidos:

```sh
git clone <SEU_REPOSITORIO_GIT>
cd <NOME_DO_PROJETO>
npm install
npm run dev
```

## Tecnologias

- Vite
- TypeScript
- React
- Tailwind CSS

## Deploy

Use seu provedor preferido (Vercel, Netlify, etc.) para publicar o projeto.
